# 📊 Walmart Sales Dashboard | Microsoft Excel

An interactive Walmart Sales Dashboard built in **Microsoft Excel** to analyze customer purchase behavior, sales performance, payment methods, product categories, repeat customers, and city-wise orders.

This project demonstrates end-to-end Excel dashboard development using **Pivot Tables**, **Pivot Charts**, **Slicers**, and **Business KPIs** to convert raw sales data into meaningful business insights.

# 📌 Project Overview

The objective of this project is to transform Walmart sales data into an interactive dashboard that enables users to monitor sales performance, customer behavior, and product trends.

The dashboard allows users to filter data dynamically using slicers and quickly identify important business insights for decision-making.

---

# 🎯 Business Objectives

- Analyze total sales performance.
- Track total customer orders.
- Monitor average customer ratings.
- Identify the best-selling product.
- Compare repeat and non-repeat customers.
- Analyze payment methods.
- Monitor monthly sales trends.
- Identify top-performing cities.
- Analyze product categories.

   - Gender
